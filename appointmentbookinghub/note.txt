
# booking system
- patients book appointments

Patient sees list of specialists
Patient clicks a specialist
Patient sees Specialist's availability calendar
Patient selects a time on Specialist's calendar
Patient sees success page


Specialist sees list of bookings


Patient {
    bookings: Booking[]
}

Specialist {
    bookings: Booking[]
}

Booking {
    patientId: Patient.id,
    specialistId: Specialist.id
}
